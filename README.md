## 12306火车票查询小工具
### 使用方法：

1. 	git clone https://github.com/pengshuang/ticket.git
2. 	cd ticket/
3.  sudo pip install requests prettytable docopt (下载项目依赖的模块）
4.  如果我要查询 2016年9月5号北京到武汉的火车票，可以使用如下命令：
	
		python tickets.py beijing wuhan 2016-09-05
